# 静态方法、类方法、实例方法
```python
def foo(x):
    print "executing foo(%s)"%(x)

class A(object):
    def foo(self, x):
        print "executing foo(%s)"%(x)
    
    @classmethod
    def class_foo(cls, x):
        print "executing class_foo(%s, %s)"%(cls, x)
    
    @staticmethod
    def static_foo(x):
        print "executing static_foo(%s)"%x

a = A()
```
类有3个方法，即静态方法（staticmethod），类方法（classmethod）和实例方法
